# dotfiles

## Setup

1. Install [brew](https://brew.sh/)
2. Clone this repo to `~/.dotfiles`
3. Install packages: `brew bundle install`
4. Symlink configs: `stow .`
5. Open Alacritty
6. Run `tmux`
7. Install tmux plugins: `prefix + I`
